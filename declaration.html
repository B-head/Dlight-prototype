ImportDeclaration:
	"import", ImportList

ImportList:
	PrefixExpression (,",", PrefixExpression)*

import math, xml

エイリアス部分的実装

AliasDeclaration:
	"alias", AliasList, PrefixExpression

AliasList:
	Identifier (,",", Identifier)*

エイリアスは、識別子の別名を宣言します。 

alias after before

変数

VariantDeclaration:
	(AttributeSignature,)? VariantPrefix, Identifier (,":", PrefixExpression)?

VariantPrefix:
	"var"
	"let"
	"const"

変数は、値を記録しておく領域です。 変数は、宣言で使用したプレフィクスによって、値を割り当てる事のできるタイミングに違いがあります。 varは、いつでも再割当て可能な変数です。 letは、宣言された場所と、変数が所属するクラスのコンストラクタでのみ値を割り当てることができます。 constは、コンパイル時にのみ値の割り当てが可能な定数で、定数に値を割り当てる式は定数式である必要があります。 

var a
let b
const c

EnumDeclaration:
	(AttributeSignature,)? "enum" (,Identifier)? (,":", PrefixExpression)? (,EnumContext)?

EnumContext:
	("::",)? ";"
	"::", EnumField, (";")?
	("::",)? "{", EnumList, "}"

EnumList:
	;? EnumField (; EnumField)* ;?
	;

EnumField:
	EnumFieldPart, ":=", NakedRangeLiteral
	NakedRangeLiteral, "=:", EnumFieldPart
	ArgumentPart

EnumFieldPart:
	(AttributeSignature,)? Identifier

enum : Int32 {
	foo := 0
	bar
	baz
	qux := 99
}

属性

AttributeSignature:
	StartAttribute, FollowAttribute*

StartAttribute:
	("@",)? EmbedAttribute
	"@", NakedRangeLiteral

FollowAttribute:
	",", Attribute
	StartAttribute

Attribute:
	StartAttribute
	NakedRangeLiteral

EmbedAttribute:
	"static"
	"override"
	"final"
	"variadic"
	"private"
	"protected"
	"public"

public static var a

アクセシビリティ属性

各宣言には、外部から参照可能な範囲を制限するためのアクセシビリティ属性を付けることができます。 publicは、参照可能な範囲を制限しません。 protectedは、その宣言が定義された子と孫を含むスコープと、クラスを継承しているクラスからのみ参照することができます。 privateは、その宣言が定義された子と孫を含むスコープからのみ参照することができます。
